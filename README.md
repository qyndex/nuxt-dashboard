# Nuxt Dashboard

Analytics dashboard with KPI cards, Chart.js DAU line chart, traffic pie chart, and events table using Nuxt 3 server routes.

**Stack:** Nuxt | **Complexity:** moderate
**Tags:** `nuxt`, `vue`, `dashboard`, `analytics`, `charts`, `admin`, `typescript`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
